# 🚀 Capstone Project: capstone-nextjstemplates

---

## 🛠 Technology Used

* **Tailwind CSS v4**
* **HTML5 / CSS3**
* **Tailwind CLI**

---

## ⚙️ Getting Started

### [01] Install Tailwind CSS v4
```Terminal
npm install tailwindcss @tailwindcss/cli
```

### [02] Start the Tailwind build process
```Terminal
npx @tailwindcss/cli -i .\src\css\input.css -o .\src\css\output.css --watch
```

### [03] Install Preline UI
```Terminal
npm i preline
```
